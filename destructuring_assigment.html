Destructuring Assigment

Destructuring assignment adalah fitur dalam JavaScript yang memungkinkan Anda untuk mengekstrak nilai dari array atau properti dari objek ke dalam variabel yang berbeda dengan cara yang lebih bersih dan singkat. Ini sangat berguna untuk membuat kode lebih mudah dibaca dan ditulis.

Contoh 1 :

  • Destructuring Array
    • Dengan destructuring array, Anda bisa mengekstrak nilai dari array ke dalam variabel dengan cara yang mudah. 
      const numbers = [1, 2, 3];

      // Destructuring Assigment
      const [first, second, third] = numbers; // bisa di loncat menggunakan ''

      document.writeln(`${first}`);
      document.writeln(`${second}`);
      document.writeln(`${third}`);

Contoh 2 :

  • Swap Items
    • Menukar posisi nilai dalam variabel. 
      let a = 1;
      let b = 2;
      document.write('Before Swap');
      document.writeln(`${a}`);
      document.writeln(`${b}`);

      document.write('After Swap');
      // swapping
      [a, b] = [b, a];
      document.writeln(`${a}`);
      document.writeln(`${b}`);

Contoh 3 :

  • Destructuring Objek
    • Dengan destructuring objek, Anda bisa mengekstrak properti dari objek ke dalam variabel. 
      const person = {
        name: 'Rio Vernando',
        age: 27,
        city: 'Nanga Pinoh',
      };

      // destructuring assigment
      const { name, age, city } = person;

      document.write(`Name : ${person.name}`);
      document.write(`Age : ${person.age}`);
      document.write(`City : ${person.city}`);

Higher Order Function

Higher order function (HOF) adalah konsep penting dalam JavaScript yang mengacu pada fungsi yang dapat menerima fungsi lain sebagai argumen, mengembalikan fungsi, atau keduanya. Fungsi-fungsi ini memungkinkan cara yang lebih abstrak dan modular dalam menangani kode, yang sering kali membuat kode lebih bersih dan lebih mudah dipahami.

Contoh 1 :

  • Map
    • Metode map membuat array baru dengan hasil pemanggilan fungsi tertentu pada setiap elemen dalam array pemanggil. 
      const arrNum = [2, 4, 6, 8];
      const kuadrat = arrNum.map((num) => num * num);
      console.log(kuadrat);

Output di console browser

Contoh 2 :

  • Filter
    • Metode filter membuat array baru dengan semua elemen yang lulus uji yang disediakan oleh fungsi yang diberikan. 
      const angka = [1, 2, 3, 4, 5, 6, 7, 8];
      const genap = angka.filter((n) => n % 2 === 0);
      console.log(genap);

Output di console browser

Contoh 3 :

  • Reduce
    • Metode reduce menerapkan fungsi terhadap akumulator dan setiap nilai dari array (dari kiri ke kanan) untuk mengurangkannya menjadi nilai tunggal. 
      const angka = [1, 2, 3, 4, 5, 6, 7, 8];
      const genap = angka.filter((n) => n % 2 === 0);
      console.log(genap);

Output di console browser

Contoh 3 :

  • Fungsi yg mengembalikan fungsi
      function greet(pesan) {
        return function (name) {
          console.log(pesan + ', ' + name + '!');
        };
      }
      const sayHello = greet('Hello');
      sayHello('Rio');
      sayHello('Rival');

Template Literal

Template literal di JavaScript adalah fitur yang diperkenalkan dalam ECMAScript 2015 (ES6) yang memungkinkan penulisan string dengan cara yang lebih mudah dan fleksibel. Template literal menggunakan backticks (`) bukan tanda kutip tunggal (') atau ganda ("), dan memungkinkan penyisipan variabel serta ekspresi langsung di dalam string.

Fitur Utama Template Literal

  • Multi-line String (String Multibaris)
  • Embedded Expression (Ekspresi yg disematkan)
  • HTML Fragments (Bagian HTML)
  • Interpolation (Interpolasi)
  • Tagged Template (Template Berpenanda)

Contoh 1 :

  • Multi-line String (String Multibaris)
    • Template literal memungkinkan penulisan string yang mencakup beberapa baris tanpa menggunakan karakter khusus seperti \n 
      let multiLineString = `
      ini multi-line string,
      didalam template literal
      javascript`;
      console.log(multiLineString);

Output di console browser

Contoh 2 :

  • Embedded Expression (Ekspresi yg disematkan)
    • Anda dapat menyisipkan ekspresi JavaScript ke dalam template literal. Ekspresi tersebut akan dievaluasi dan hasilnya akan dimasukkan ke dalam string 
      let a = 5;
      let b = 10;

      let result = `${a} + ${b} = ${a + b}`;
      console.log(result);

Output di console browser

Contoh 3 :

  • HTML Fragments (Bagian HTML)
    • Memungkinkan kita untuk dapat menyisipkan bagian HTML ke dalam template literal `` 
      const mhs = {
        nama: 'Rio Vernando',
        nim: 145410169,
      };

      let element = `
      

        
${mhs.nama}

        ${mhs.nim}
      
`;
      console.log(element);

Output di console browser

Contoh 4 :

  • Interpolation (Interpolasi)
    • Dengan template literal, Anda dapat menyisipkan variabel dan ekspresi langsung ke dalam string menggunakan ${} 
      let name = 'Rio';
      let age = 27;

      let message = `Hello my name is ${name}, i'am ${age} old`;
      console.log(message);

Output di console browser

Contoh 5 :

  • Tagged Template (Template Berpenanda)
    • Template tagged memungkinkan Anda untuk memanggil fungsi dengan literal template sebagai parameter. Ini berguna untuk manipulasi string yang lebih kompleks, seperti sanitasi input atau penerjemahan 
      function tag(strings, ...values) {
        console.log(strings);
        console.log(values);

        return strings[0] + values[0] + strings[1] + values[1] + strings[2];
      }

      let person1 = 'Evan';
      let person2 = 'Noven';

      let taggedResult = tag`Hello ${person1} and ${person2} are friends.`;
      console.log(taggedResult);

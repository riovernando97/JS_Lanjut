Destructuring Function

Destructuring function adalah teknik dalam JavaScript yang memungkinkan Anda untuk mengekstrak nilai dari objek atau array langsung sebagai parameter dari fungsi. Ini memungkinkan Anda untuk menyederhanakan sintaksis saat menggunakan objek atau array sebagai argumen, dan memungkinkan Anda untuk langsung mengakses nilai-nilai yang relevan tanpa perlu merujuk ke objek atau array secara langsung di dalam tubuh fungsi.

Contoh 1 :

  • Destructuring Objek sebagai Parameter Fungsi
    • Anda bisa menggunakan destructuring untuk mengekstrak nilai dari objek sebagai parameter fungsi. 
      function printPerson({ name, age, address: { city, street, province } }) {
        // destructuring
        document.writeln(`Name : ${name}, Age : ${age}, and i live in ${city}`);
      }

      const person = {
        name: 'Rio Vernando',
        age: 27,
        address: {
          city: 'Nanga Pinoh',
          street: 'Jl. Serundung Permai II',
          province: 'Kalimantan Barat',
        },
      };

      printPerson(person);

      // Contoh lain
      function tambahKali(a, b) {
        return [a + b, a * b];
      }

      const [jumlah, kali] = tambahKali(2, 3); // destructuring
      document.writeln(`Penjumlahan = ${jumlah}`);
      document.writeln(`Perkalian = ${kali}`);

Contoh 2 :

  • Destructuring Array sebagai Parameter Fungsi
    • Destructuring juga bisa digunakan untuk mengekstrak nilai dari array sebagai parameter fungsi. 
      function printCoordinate([x, y]) {
        document.writeln(`Coordinates : ${x}, ${y}`);
      }

      const point = [10, 20];
      printCoordinate(point);

Contoh 3 :

  • Penggunaan Default Values
    • Seperti dalam destructuring pada umumnya, Anda juga bisa menyediakan nilai default untuk parameter fungsi yang mungkin tidak ada nilainya. 
      function greetUser({ name = 'Guest', age = 30 }) {
        console.log(`Hello, ${name}! You are ${age} years old.`);
    }
    
    const user = {
        name: 'Alice'
    };
    
    greetUser(user); // Output: Hello, Alice! You are 30 years old.

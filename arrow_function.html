ARROW FUNCTION

Arrow function adalah sintaks ringkas untuk mendefinisikan fungsi di JavaScript yang diperkenalkan dalam ECMAScript 6 (ES6). Arrow function menawarkan cara yang lebih pendek untuk menulis fungsi anonim (fungsi tanpa nama) dan memperkenalkan perilaku leksikal untuk this yang berbeda dari fungsi biasa.

Contoh 1 :

      // 1 parameter, tidak perlu kurung diparameter (), tidak perlu return dan kurung kurawal
      // (implisit return)
      let tampilNama = (nama) => `Halo ${nama}`;
      console.log(tampilNama('Rio'));

Output di console browser

Contoh 2 :

      // tampilkan dalam bentuk array
      let car = ['Toyota', 'Mistubishi', 'Suzuki', 'Honda'];

      let arrLength = car.map((merk) => merk.length);
      console.log(arrLength);

Output di console browser

Contoh 3

      // tampil dalam bentuk objek
      let handphone = ['Nokia', 'Samsung', 'Oppo'];

      let hpLength = handphone.map((brand) => ({ nama: brand, jmlHuruf: brand.length }));
      console.table(hpLength);

Output di console browser

Contoh 4 :

      // > 1 parameter
      let greet = (name, time) => {
        return `Selamat ${time}, hai ${name}`;
      };
      console.log(greet('Evan', 'siang'));

Output di console browser

Contoh 5 :

'this' dalam arrow function

      // arrow function (this)
      function Mhs(nama) {
        this.nama = nama;
        this.sapa = () => {
          console.log(`Halo ${this.nama}`);
        };
      }

      const orang = new Mhs('Aling ');
      orang.sapa();

Output di console browser

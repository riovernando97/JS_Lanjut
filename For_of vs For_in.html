For Of vs For In (Loop)

  • For Of :
    • for...of digunakan untuk mengulangi atau "loop" melalui elemen-elemen dalam sebuah objek yang bisa diiterasi, seperti array, string, Map, Set, dan lainnya. Ini memungkinkan Anda mengakses nilai-nilai langsung, tanpa perlu mengakses indeksnya. Ini lebih mudah dibaca dan digunakan saat Anda hanya membutuhkan nilai tanpa memikirkan indeksnya.

          // Contoh 1
      const fruits = ['Apel', 'Banana', 'Orange'];

      for (const fruit of fruits) {
        document.writeln(`List Fruits : ${fruit}`);
      }

      // Contoh 2
      const numbers = [1, 2, 3, 4, 5];

      for (const num of numbers) {
        if (num % 2 === 0) {
          document.writeln(`${num} = Genap`);
        } else {
          document.writeln(`${num} = Ganjil`);
        }
      }

      // Contoh 3
      const mhs = ['Rio', 'Rival', 'Evan'];

      for (const [i, m] of mhs.entries()) {
        document.writeln(`${m} adalah mahasiswa ke - ${i + 1}`);
      }

      // Contoh 5 (Arguments)
      function jmlAngka() {
        let jumlah = 0;
        for (a of arguments) {
          jumlah += a;
        }
        return jumlah;
      }
      document.writeln(jmlAngka(1, 2, 3, 4, 5));

  • For In
    • for...in juga digunakan untuk melakukan "loop" melalui properti-properti sebuah objek, namun, ia lebih cocok digunakan untuk objek yang memiliki properti enumerable (dapat dihitung). Ini biasanya digunakan untuk objek JavaScript biasa, bukan untuk array atau tipe data terurut lainnya. 
          const person = {
        name: 'Rio Vernando',
        age: 27,
        email: 'riojr997@gmail.com',
      };

      for (const key in person) {
        document.writeln(`${key}: ${person[key]}`);
